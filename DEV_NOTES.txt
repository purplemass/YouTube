------------------------------------------------------------------------
Installation on Windows
------------------------------------------------------------------------
1 Download Python 2.7.1 [Windows x86 MSI Installer (2.7.1)] from
  http://www.python.org/download/releases/2.7.1/

2 Install with default settings

3 Download Youtube's gdata-python-client [gdata-2.0.14.zip] from
  http://code.google.com/p/gdata-python-client/downloads/detail?name=gdata-2.0.14.zip&can=2&q=

4 Unzip file to the root of the C drive so a new folder is created here: C:\gdata-2.0.14\

5 Set the PYTHONPATH environmental variable to C:\gdata-2.0.14\src\ (or add to it if variable exists):
  - Go to Control Panel -> System-> Advanced.
  - Click 'Environment Variables' button.
  - In the 'System Variables' panel that appears, click New and enter PYTHONPATH in the 'Variable Name' field.
    In the 'Variable Value' field enter path to C:\gdata-2.0.14\src\

------------------------------------------------------------------------
Installation on OSX
------------------------------------------------------------------------
1 Download Youtube's gdata-python-client [gdata-2.0.14.zip] from
  http://code.google.com/p/gdata-python-client/downloads/detail?name=gdata-2.0.14.zip&can=2&q=

2 Unzip file to a known location e.g. /gdata-2.0.14/

5 Set the PYTHONPATH environmental variable to /gdata-2.0.14/src/:
  - PYTHONPATH="/Users/purplemass/Downloads/gdata-2.0.14/src/gdata/:$PYTHONPATH"
  - export PYTHONPATH

------------------------------------------------------------------------
Running the application
------------------------------------------------------------------------
Windows: Double-click main.py

OSX:
- Open Terminal
- Change directory to the application folder
- Type: python main.py and press Enter

------------------------------------------------------------------------
Stopping the application
------------------------------------------------------------------------
Hold down the Control key and press C (<CTRL> + C)
